“Everything — from biological life to inanimate systems — generate shape and structure and evolves in a sequence of ever-improving designs in order to facilitate flow.
/ / / / / / / / / / / / / / / / / / / / / / / / / / / / / / / / /
River basins, cardiovascular systems, and bolts of lightning are very efficient flow systems to move a current — of water, blood, or electricity.
/ / / / / / / / / / / / / / / / / / / / / / / / / / / / / / / / /
Likewise, the more complex architecture of animals evolves to cover greater distance per unit of useful energy, or increase their flow across land.
/ / / / / / / / / / / / / / / / / / / / / / / / / / / / / / / / /
Such designs also appear in human organizations, like the hierarchical ‘flowcharts’ or reporting structures in corporations and political bodies.”
